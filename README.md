🚀 Projects Overview

Task 2: BMI Calculator 🏥

A comprehensive BMI (Body Mass Index) tracking application with data persistence and visualization.



Features:



Multi-user support with user registration.



BMI calculation with health category classification (Underweight, Normal, Overweight, Obese).



Historical data tracking with SQLite database.



Interactive BMI trend charts using Matplotlib.



Modern dark-themed GUI with PyQt6.



Technologies: Python 3, PyQt6, SQLite, Matplotlib.



How to Run:



Bash

cd "Task2-BMICalculator"

pip install PyQt6 matplotlib

python main.py
